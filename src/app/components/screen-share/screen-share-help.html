Screen sharing

Sharing your screen is an experimental feature in WebRTC and only works if the browser you're using meets particular requirements.

Active screen/window sharing will only work from Firefox if you have at least version 33 installed. In addition, two adjustments must be done in the settings at about:config. First, make sure media.getusermedia.screensharing.enabled is enabled. Then, make sure {{ vm.hostname() }} is listed in media.getusermedia.screensharing.allowed_domains.

If you're using a recent version of Chrome (34+), you'll need to install this extension. If you're using an older verson of Chrome, instead, you'll need to manually enable the Chrome flag chrome://flags/#enable-usermedia-screen-capture. Otherwise you'll get a permission error. Moreover, please beware that some versions of Chrome may only support the feature if you explicitly pass the --enable-usermedia-screen-capturing parameter on the command line.
