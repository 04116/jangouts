Information about browsers

Firefox

At least Firefox 36 is required, a more recent version is recommended.

Firefox will refuse to use any bit rate lower than a certain limit (200 kbps by default). In some situations, a lower bit rate can be useful to save bandwidth and processing power in the clients. This limit can be adjusted (a sensible value would be 64) modifying the media.peerconnection.video.min_bitrate setting at about:config.

Chrom(e/ium)

Jangouts has been successfully tested with Chromium 41. It will probably work with older versions as well, but the usage of a version as recent as possible is recommended.

Under certain circumstances, some versions of Chromium has been reported to cause an excessive CPU usage in the Janus server powering Jangouts. This problem has not been observed with recent versions of Janus and Chromium.
